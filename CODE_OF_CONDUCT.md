# Code of Conduct

## Our Pledge

We want this project to stay welcoming, respectful, and easy to contribute to.

## Expected behavior

- Be respectful and constructive.
- Focus on the code, results, and ideas.
- Give clear feedback.
- Assume good intent.
- Keep discussions helpful and on topic.

## Unacceptable behavior

- Harassment or abuse
- Personal attacks
- Discriminatory language
- Deliberate disruption
- Repeated off-topic hostility

## Enforcement

Maintainers may remove content, close issues, or reject contributions that violate this code of conduct.
